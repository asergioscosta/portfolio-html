Java é uma linguagem de programação lançada pela Sun Microsystems em 1995. Existem muitos aplicativos e sites que não funcionarão a menos que você tenha o Java instalado.   • O que é Adobe Photoshop?

Adobe Photoshop é um dos aplicativos de software de edição de fotos e gráficos mais populares e conhecidos do mercado. Foi desenvolvido em 1988 e desde então se tornou um pilar para usuários que desejam editar fotos existentes ou até mesmo criar suas próprias imagens e gráficos. Microsoft Visual Studio é um ambiente de desenvolvimento integrado (IDE) da Microsoft para desenvolvimento de software especialmente dedicado ao .NET Framework e às linguagens Visual Basic (VB), C, C++, C# (C Sharp) e F# (F Sharp).
